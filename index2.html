 New Media Design and Web Developmentの3ターム目のスケジュール

Portfolio

４ターム目のプラクティカムに向けて個人で最低６つプロジェクトを考えてポートフォリオウェブサイトを作成して公開します。ポートフォリオ自体はプロジェクトを載せるウェブサイトであれば結構なんでもありです。（ワードプレスの無料テンプレートでもw）基本的に自分の好きな分野（このコースでいうとウェブマーケティング、映像、プログラマー、デザイナー）にフォーカスして６つのプロジェクトの内容を考えます。自分のポートフォリオはまたの機会に載せようと思います！

Project Managemnet

このコースではプロジェクトマネージメント（簡単にいうと大人数でのウェブサイト制作）の流れについて学習します。グループにごとに別れて1つのウェブサイトを作るという課題がタームを通して与えられます。大きく分けるとプロジェクトマネージャー、コンテントマネージャー、デザイナー、プログラマーという職務をチームで分業して進めて行く形になります。

Social Networking

基本的にはマーケティングクラスでソーシャルネットワーク、SEOについて簡単に学ぶクラスです。これまたグループワークとしてウェブサイトを作成してソーシャルネットワークを使ってブランディングするとういう課題がタームを通してあります。

Web Scripting 2

ターム２ではJavascriptを勉強しましたが、今回はPHPに移ります。最終課題はログイン、ログアウトできる掲示板を作ります。しかし、前回のタームでもあったように教える先生によって難しさが違います。このプログラムでは２クラスに別れて同じカリキュラムをこなしているのですが他クラスと自分のクラスを比べると難しさに天と地ほどの違いがありました。（ちなみに自分のクラスは難しい方）

Web Design and Interface

フロントエンド全般の基礎的な知識について学びます。大きく学んだことをリスト化すると

  • Sass(GUIコンパイルソフトウェアを使います)
  • CSS Grid
  • Flex Box
  • Bootstrap
  • さらっとjQuery
  • jQuery プラグイン

主にCSSが多かった印象です。

Law and New Media

名前の通り法律です（笑）英語がめちゃくちゃ難しいですがクラスメートと協力してどうにか終わらせることができました。ひたすら座学です。眠すぎて死にました....。でもプラスで英語の勉強になると思います。

Visual Effect

Affter Effectの基礎を学ぶクラスです。アニメーションやエフェクトをかけたりします。特に強いこだわりがなければ授業についていけばなんとかなる感じでした。

Advanced Photoshop and Illustrator

フォトショとイラレを組み合わせて使うといった感じのクラスですが、中身はUX/UIを全般としていて最後の課題はSketchを使うという落ちでした（先生によります）。他のクラスの人は本当にフォトショとイラレしか使えずひたすら基本的なことを強いられていると愚痴をこぼしてました...

3ターム目を終えて

３ターム目までくるとできる人とできない人の差がはっきりしてきます。特にPHPはついていけない人続出でクラスメイトの１人はできる子にお金を払ってやってもらったりするという始末....（笑）と言ってもやってることは基礎なので普通に勉強してればついていけないこともないです。全体的に見ると全員最初ほどのモチベーションもなく、ゆるくなっています。ってなわけで生徒同士も仲良くなっていて卒業が悲しい（T_T) 笑。（もちろん自分の英語力に関しては未だ大きな問題です。）

フロントエンドを学んでいる者からの観点

４ターム目どうなるかはわかりませんが、今のところフロントエンドで必須である３代フレームワーク(React, Vue, Angular)一言も触れませんwww

ウェブサイトを見ると４ターム目はワードプレスを主に使ったサイト開発みたいなのでおそらくこれからも出てくることはないと思います。なのでフロントエンド未経験でこのコースを取るとなるとその辺は自学しないといけないです。おそらくですが他のスキル（デザイン、映像）も初心者に毛が生えた程度です。ってことでこのコースを考えてる方は学校で学べることに関しては過度な期待しない方がいいです（・＿・)v

 

卒業後のビザの為に頑張ろう（笑）
